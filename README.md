 📊 Retail Sales Analysis Dashboard

📌 Project Overview

This project analyzes retail sales data using SQL, Python, Excel, and Power BI to extract meaningful business insights and build an interactive dashboard.

---

📁 Dataset

* 5 retail transactions (Jan 2025)
* Cities: Mumbai, Pune, Nagpur
* Total Revenue: ₹2,26,000

---

🛠 Tools Used

* SQL (MySQL)
* Python (Pandas, Matplotlib)
* Excel
* Power BI

---

📈 Key Insights

* 💰 Total Revenue: ₹2,26,000
* 🥇 Laptops generate highest revenue (~66%)
* 🏙 Mumbai & Pune are top-performing cities
* 📊 Electronics dominate (~93% of total sales)
* 🎧 Headphones sold most units

---

 📊 Dashboard Features

* KPI Cards (Revenue, Orders, Cities, Products)
* Sales by Product (Bar Chart)
* Sales by City (Bar Chart)
* Category Distribution (Pie Chart)
* Sales Trend Over Time (Line Chart)

---
📂 Project Files

* `sales.csv` → Dataset
* `Retail_Sales_Project.ipynb` → Python analysis
* `sql_queries_project.sql` → SQL queries
* `excel_analysis_project.xlsx` → Excel analysis
* `project_dashboard.pbix` → Power BI dashboard
* `Capstone_Report.pdf` → Final report

---

🚀 Conclusion

This project demonstrates end-to-end data analysis using multiple tools and highlights how data-driven insights can support business decision-making.
